Build iOS apps directly in your browser on Windows and Linux platforms. iCodeWin provides a complete web-based IDE for iOS app development without requiring a Mac.

## üìã Overview

iCodeWin revolutionizes iOS app development by eliminating the need for Mac hardware. This browser-based IDE allows developers on Windows and Linux to create, test, and build iOS applications with a live preview that updates in real-time as you code.

The platform supports HTML, CSS, and JavaScript development with instant visual feedback, making it ideal for rapid prototyping and development of iOS applications across platforms previously locked out of the Apple ecosystem.

## ‚ú® Features

- **Cross-platform iOS Development** - Build iOS apps from any device with a modern web browser
- **Live Preview** - See your changes instantly with a real-time iPhone preview
- **Full-screen WebView .ipa Generation** - Export production-ready iOS app packages
- **Push Notification Support** - Implement and test scheduled push notifications
- **Universal Signing Compatibility** - Works with all signing services, including AltStore
- **In-browser Code Editor** - Complete development environment with syntax highlighting
- **File Management** - Create and organize project files directly in the browser

## üöÄ Getting Started

### Installation

No installation required! iCodeWin runs entirely in your web browser.

Simply visit:
```
https://icodewin.vercel.app/
```

### Usage

1. **Access the Platform**
   - Open your web browser and navigate to [https://icodewin.vercel.app/](https://icodewin.vercel.app/)

2. **Create a New Project**
   - Review the release notes
   - Click "New Project" to begin

3. **Develop Your App**
   - Edit the default HTML, CSS and Javascript

4. **Preview Your App**
   - Use the live iPhone preview panel to see your changes in real-time
   - Test functionality as you develop

5. **Export Your App**
   - Click "Generate .ipa" when your app is ready
   - Download the .ipa file for distribution or testing
